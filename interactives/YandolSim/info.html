Basic Information

Yandelere Simulator is a game about Cowombia High School. It should be noted, by the way, that any resemblance to real life characters is mostly entirely coincidental.

PLEASE NOTE: it often takes ~1-2 MINUTES to load the game.

Playing on mobile: You can play on mobile if, in the title screen, you enable "mobile mode". Only Android devices can run this; Apple requires all browsers on IOS effectively are Safari, reskined (in the US, anyways) man I hate Apple. This matters because of Safari's terrible standards compliance; so this game can't run on IOS, sorry.
